System Overview

WH2 Radio is an AI-powered music discovery platform.

The purpose of WH2 is to discover, organize, review, approve, and promote AI-assisted creators and independent artists through a radio-style listening experience.

The WH2 system currently consists of:

  • Public Homepage
  • Music Submission Form
  • Owner Login
  • Owner Dashboard
  • Submission Review Queue
  • Music Library
  • Artist Database
  • Legal Pages
  • Operations Manual

Daily Operations

  1. Login to WH2 Owner Dashboard
  2. Review new music submissions
  3. Approve or reject submissions
  4. Verify approved songs appear in the library
  5. Verify artist records appear in the artist database
  6. Update featured creator if desired
  7. Update station message if needed
  8. Verify station status is ON

Music Review Process

  • Open Music Submissions
  • Review song title
  • Review artist information
  • Review genre
  • Review submission type
  • Review AI tool information if provided
  • Open song link
  • Open cover art link
  • Determine approval status
Never approve music that appears stolen, fraudulent, misleading, or copyrighted by someone else.

Artist Approval Process

  • Approve submission
  • System sends approved song into Music Library
  • Artist automatically appears in Artist Database
  • Artist becomes eligible for Featured Creator status

Featured Creator Process

  • Open Artist Database
  • Select approved creator
  • Set creator as Featured Creator
  • Verify featured creator displays correctly

Station Message Process

  • Open Dashboard
  • Edit Station Message
  • Click Save Message
  • Verify message appears correctly on homepage

Library Management

  • Approved songs are stored inside Music Library
  • Review library periodically
  • Remove duplicate records
  • Remove broken links
  • Remove copyrighted content when necessary

Copyright Complaint Process

  • Receive complaint
  • Document complaint details
  • Locate artist record
  • Locate approved song
  • Temporarily remove content
  • Investigate ownership claim
  • Restore or permanently remove content

Training New Staff

  • Provide dashboard access
  • Review operations manual
  • Train on submission reviews
  • Train on artist management
  • Train on copyright procedures
  • Train on station operations

Emergency Procedures

  • Station not loading → Check dashboard status
  • Player offline → Check Live365 status
  • Database issue → Check Firebase Console
  • Login issue → Verify owner account access
  • Broken links → Remove affected records
  • Copyright claim → Temporarily remove content immediately
