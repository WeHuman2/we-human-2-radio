WH2 Radio Owner Instructions

Public Radio Player

Open WH2 Radio Player

This is what listeners use to hear WH2 Radio, view the featured artist, and submit music.

Owner Login

Open Owner Login

Use the WH2 owner email and password to access the private dashboard.

Owner Dashboard

The dashboard lets you manage the radio station.

  • Turn WH2 Radio ON or OFF
  • Update the current song title
  • Update the artist name
  • Update the MP3 file name
  • Update the cover image file name
  • Update the Featured Artist section
  • Review music submissions
  • Open the Approved Songs Library
  • Share the public player link
  • Open legal pages

Featured Artist

Use the Featured Artist card in the dashboard to update:

  • Featured artist name
  • Featured artist bio
  • Featured artist image
  • Featured artist link

Artist Music Submissions

Artists can submit music through the public submission page.

Open Music Submission Page

Artists can submit:

  • Artist name
  • Song title
  • Email
  • Music link
  • Cover art link
  • Notes
  • Permission agreement

Review Submissions

In the dashboard, click Open Submissions.

From there, you can:

  • View new submissions
  • Open submitted music links
  • Open cover art links
  • Approve submissions
  • Reject submissions

Approved Songs Library

Open Approved Songs Library

The Approved Songs Library stores approved tracks for WH2 Radio.

You can:

  • Add approved songs manually
  • View approved songs
  • Delete songs from the library
  • Make a song the current live track

Auto Playlist

The public player now pulls music from the Approved Songs Library.

If multiple songs are approved, the player can move to the next song automatically when the current song ends.

How to Add New Music Manually

  1. Upload the MP3 file to GitHub.
  2. Copy the exact MP3 file name.
  3. Go to the dashboard.
  4. Paste the file name into MP3 File Name.
  5. Enter the song title and artist name.
  6. Click Save Song Details.

How to Change Cover Art

  1. Upload the cover image to GitHub.
  2. Copy the exact image file name.
  3. Paste it into Cover Image File Name.
  4. Click Save Song Details.

Important Notes

  • Keep the owner login private.
  • Do not approve music unless the artist has permission to submit it.
  • Music links should be reviewed before adding songs to the library.
  • Future upgrades can include analytics, artist profile pages, and direct file uploads.
