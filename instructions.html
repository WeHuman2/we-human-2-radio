Turn Radio On/Off

Log in, open the dashboard, then use TURN RADIO ON or TURN RADIO OFF.

Change Song Info

In the dashboard, update Song Title, Artist Name, and MP3 File Name. Then click SAVE SONG INFO.

Add New Music

Upload the MP3 file to GitHub, copy the exact file name, paste it in MP3 File Name, then save.
